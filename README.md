# Kincony A8 - 14 Inputs Extender 

> **_NOTE:_**  **The address of the MCP is fixed to 0x20**

**BOM is not created in the files**

3v3 - TSR2-2433N - TracoPower

For TCMT1100 Anod connection i used 1.8K Ohm with 12VDC input - 0805

For TCMT1100 Emitter connection i used 4.7K Ohm for pullup on 3v3. - 0805

Capacitor 100nF - 1026

2 x Connector 1x7 - 1823683 - PhoenixContact


> **_NOTE:_**  **There are no I2C resistors on the board, if this is used for somethig else they must be placed somewhere else if needed**
